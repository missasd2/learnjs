js 只有一种数字类型；

所有js数字均为64位

js不定义不同类型的数字，不如整数、短、长、浮点等等。

js不区分浮点类型和整数类型；所有的数字都是浮点型

精度

不使用小数点和科学计数法的情况下，数字最多有15位。

八进制和十六进制

如果前缀为0, 比如 var y = 001; 则y被解释为八进制数，其对应十进制的值为8

如果前缀为0x, 比如 var z = 0x0F; 则y被解释为十六进制数，其对应十进制的值为15

无穷大 (Infinity)

当数字超出上下限，则结果为 Ifinity。

NaN-非数字值

NaN 与 undefined的区别

  • (1) NaN 的类型为 number，undefined的类型为undefined

数字可以是Number类型如 var x = 123; 也可以是Object类型, 如 var y = new Number(123);
